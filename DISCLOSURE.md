# Generative AI Disclosure

In accordance with academic integrity guidelines, course regulations, and best practices for algorithmic transparency, this document discloses the use of Generative Artificial Intelligence (GenAI) technologies in the development and documentation of the **User Control in Tourism Recommender Systems** project.

## Technologies Used
- **Model**: Google DeepMind's Gemini 3.5 Flash
- **Interface**: Antigravity agentic coding assistant

## Scope of Assistance

### 1. Software Development
Generative AI was utilized as a pair-programming assistant to:
- **UI & Layout Design**: Generate structure and styling parameters in HTML and CSS, specifically establishing the glassmorphism aesthetic (translucent cards, gradients, and micro-interactions).
- **Algorithmic Debugging**: Analyze and propose corrections for mathematical singularities in the client-side JavaScript engine (specifically handling division-by-zero when total slider weights equal zero, and preventing singularities in relative price/location normalization when $x_{\text{max}} = x_{\text{min}}$).
- **Component Optimization**: Advise on React hooks (such as `useMemo` and `useEffect`) to ensure efficient state propagation and real-time computation of recommendations.

### 2. Documentation and Writing
Generative AI assisted in the drafting of the scientific report (`paper/paper.tex`):
- **Formatting and Templates**: Adapting the boilerplate ACM document class (`acmlarge`) to resolve formatting incompatibilities (e.g., removing undefined commands like `\correspondingauthor` and correcting ampersand escaping in headings).
- **Drafting and Synthesis**: Helping draft and refine the text of the report based on project design documents, presentation materials, and implementation notes. This includes the formulation of mathematical equations (MAUT and Constraint-Based Filtering) and the creation of formatted tables (hotel database and contribution records).

## Human Audit and Verification
All output generated or suggested by Generative AI was reviewed, audited, and approved by the human author (David Gallob). The final mathematical formulations, logic changes in the software, experimental designs, and report text represent the verified academic work of the author.
